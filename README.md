This repository contains a complete SQL learning journey based on a structured syllabus, covering fundamental to advanced database concepts using MySQL.

It is designed to provide hands-on practice and real-world understanding of SQL through progressive topics, examples, and exercises.

## 📚 Topics Covered

* Database Fundamentals
* Data Definition Language (DDL)

  * CREATE, ALTER, DROP
* Data Manipulation Language (DML)

  * INSERT, UPDATE, DELETE
* Data Query Language (DQL)

  * SELECT, WHERE, ORDER BY, GROUP BY, HAVING
* Constraints

  * PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL, CHECK, DEFAULT
* Joins

  * INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN
* Subqueries
* Aggregate Functions

  * COUNT, SUM, AVG, MIN, MAX
* Indexes
* Views
* Stored Procedures & Functions
* Triggers
* Transactions

  * COMMIT, ROLLBACK, SAVEPOINT
* Normalization
* Error Handling & Debugging

## 🎯 Objective

To build a strong foundation in SQL and database management by practicing structured queries and solving real-world problems.

## 🚀 Outcome

By completing this repository, the learner will be able to:

* Design and manage databases
* Write efficient SQL queries
* Understand relationships between tables
* Handle real-time database operations confidently

---

This repository will be continuously updated with new topics, practice sets, and mini-projects as part of an ongoing SQL learning journey.
